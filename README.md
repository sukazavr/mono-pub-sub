# mono-pub-sub

> Dead simple implementation of the Publish/Subscribe pattern

## Key features

- Dependency free
- ES6/CommonJS module support
- Easy to understand and use (thanks to prototype pattern)
- Small(ish), less than 1kb minified and gzipped
